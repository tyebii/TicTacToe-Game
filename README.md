# Tic Tac Toe Game

A simple text-based tic tac-toe-game that supports multiple clients and game instances.

## Features
- Server code for hosting multiple concurrent game instances and servicing multiple user connections (per game)
- Each game has its own unique join code, with a maximum of 2 players per game
- Simple interface with commands for playing the game and an ASCII art board display
